To run mltest you must specify config_filename (-f) and data_path (-d).
config_filename  -  full name of file OpenCV/opencv_extra/testdata/ml/validation.xml
data_path  -  full path to dataset files - OpenCV/opencv_extra/testdata/ml

For example
mltest.exe -f C:/OpenCV/opencv_extra/testdata/ml/validation.xml -d C:/OpenCV/opencv_extra/testdata/ml
